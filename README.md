# nginx-cdn
CentOS一键安装Nginx

### 环境要求
* CentOS 6/7

### 安装
```
wget https://raw.githubusercontent.com/helloxz/nginx-cdn/master/nginx.sh && bash nginx.sh
```

* 如果是CentOS 6的系统，请选择编译安装。
* CentOS 7可选择编译安装或二进制安装。
* 编译安装通常10分钟左右，二进制安装通常2分钟左右。
